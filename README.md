# lms-demo-client

## Project setup
```
yarn install
```
- - - -
### Create a `.env` File  or Copy `.env.expample` File and rename it to `.env` 
- - - -
### Add your Backend URL `.env` FILE
`VUE_APP_BACKEND_URL=YOUR_BACKEND_URL`
- - - -
### Production Build
```
yarn build
```

- - - -

### Compiles and hot-reloads for development
```
yarn serve
```
